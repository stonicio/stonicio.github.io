Stonic Project

Welcome to Stonic Project!

Stonic is an agentless deployment tool similar to Ansible.

Current state

Stonic is in early development state now. It is clear that several first Stonic releases will maintain compatibility with Ansible.

Subscribe to the Stonic Blog for all the latest news about the project state and development.

There is Stonic Project Board where you can track the current state of the development and all future plans and design ideas.

How to contribute

At this stage we encourage you to file anything you wish Stonic should have or something you'd like to see in Ansible that is not there using the Stonic Issue Tracker.
